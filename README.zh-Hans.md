## 项目介绍

electron-vite + inversify + trpc-electron 启动模板

## 功能介绍

- 像Nestjs一样写后端代码
- 使用tRPC，您可以轻松构建和使用完全类型安全的API，而无需模式或代码生成

## 安装

```sh
git clone https://github.com/shilim-developer/electron-trpc-inversify
```

```sh
cd electron-trpc-inversify
```

```sh
pnpm install
```

```sh
pnpm run dev
```
